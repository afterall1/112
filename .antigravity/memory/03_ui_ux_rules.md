# UI/UX Rules

## Design Philosophy
**"Darkness with Purpose"** - The UI should feel like a HUD on a spaceship deep in the void.

## Color Palette

### The Void
*   **Deep Space:** `#050505` (Background)
*   **Stardust:** `#ffffff` (High contrast text, low opacity for stars)

### Entities
*   **Neon Cyan (Risers):** `#00f3ff`
    *   *Role:* Ascension, Positive Momentum, Velocity.
*   **Neon Magenta (Fallers):** `#ff0055`
    *   *Role:* Gravity, Singularity, Negative Momentum.

## Typography
*   **Font Family:** `'Rajdhani', sans-serif`
    *   *Weights:* 500 (Medium) for body, 700 (Bold) for headers.
    *   *Vibe:* Technical, squared-off, futuristic.

## Animation & Physics
*   **Easing:** `cubic-bezier(0.16, 1, 0.3, 1)` (Expo Out for snappy tech feel).
*   **Motion Rules:**
    *   Positive % change -> Outward velocity vector.
    *   Negative % change -> Inward gravity vector.
